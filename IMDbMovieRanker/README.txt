# IMDb Movie Ranker

IMDb Movie Ranker is a simple Python project that allows users to fetch the IMDb rating of a movie by entering its name. The project utilizes the IMDbPY library to interact with the IMDb database and retrieve movie details.

## Features

- User inputs the movie name.
- Fetches the IMDb rating for the specified movie.
- Displays the rating in the console.

## Prerequisites

Before running the project, make sure you have the following installed:

- Python 3.x
- IMDbPY library

You can install the required library using pip:

```bash
pip install IMDbPY


How to Run:

1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Run the Python script:

```bash
python IMDbMovieRanker.py

4. Enter the movie name when prompted.
Example:

Please enter the movie name: Twisters
The IMDb rating for Twisters is 6.7.
