# TextToSpeech-English

This is a simple Python program that converts text input from the user into speech using the `gTTS` (Google Text-to-Speech) library.

## Features

- Convert any English text into speech.
- Saves the speech as an MP3 file.
- Automatically plays the MP3 file after saving.

## Prerequisites

Before running the program, you need to install the required Python package. You can install `gTTS` by running the following command:

```bash
pip install gTTS

python TextToSpeech.ipynb
